詳解

BCTの実行周りの関数, パラメーター

覚え書き
Block Command ExeInfoは Block Command Table (BCT) の実行側のパラメーターである. 現状 Cmd のデータ保存側と実行側は分離されておらず, BCT のその実行状態はここに保存される. よって実行パラメーターは BCT と同数だけ (BCT_MAX_BLOCKS) 確保される.
