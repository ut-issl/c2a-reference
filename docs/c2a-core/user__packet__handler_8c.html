PH_ACK PH_user_analyze_cmd ( const CTCP *  packet)

    Coreの処理より優先される
    C2Aを搭載したコンポのキュー（GSC，RTC，TLC，BC）に即座に直接追加される．

    つまり，本OBCにとってはRTCだが，転送対象OBCにとっては各種Cmd扱いなCmdを転送する処理
引数
*packetCTCP packet
戻り値

PH_PL_LIST_FULLPL LIST FULL
