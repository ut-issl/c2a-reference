詳解

アノマリやエラーなどの各種イベントを記録する

覚え書き
このイベントをもとに event_handler を発火させることができる
ログは TLog (TimeSeriesLog) と CLog (EL_CumulativeLog) の二種類を準備し，必要なもののみを使う
今後， NvLog (不揮発Log) が実装予定
詳細は event_logger.h を参照

event_logger.c に定義があります。
