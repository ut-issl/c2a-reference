◆ PL_PACKET_TYPE

enum PL_PACKET_TYPE

PacketList に格納される packet の型情報

覚え書き
uint8_t を想定
基本的に PacketList は任意の packet 型を格納できるようにしているが，特定のメソッドが特定の型を要求するため，作っている 以下で定義されている型については，それを指定して PL_initialize を呼び出すことを推奨する（内部でアサーションなどもかかるので）
列挙値
PL_PACKET_TYPE_CTCP

PL_PACKET_TYPE_CTP 

CommonTlmPacket

PL_PACKET_TYPE_CCP
