コンポ間通信などで標準的に使う ネットワーク層 の EB90 Packet

覚え書き
一般的には，データリンク層は EB90 Frame を使うことを想定
Tlm か Cmd かはコンテキストで読み替える
データリンク層は DS_StreamConfig.data_link_layer_ で規定する
